問13RFI

提案前にベンダから情報を集める文書を見分ける

RFIは情報提供依頼、RFPは提案依頼として、調達の前段階で市場や製品情報を集める役割を整理します。

1結論を先に押さえる

RFIは Request For Information の略で、情報提供依頼書です。発注側がベンダに製品、サービス、技術、実績などの情報提供を求める文書です。

2RFIとRFPを分ける

  • RFI：情報提供依頼。要件を固める前に市場や候補ベンダの情報を集める。
  • RFP：提案依頼。要求や条件を示し、具体的な提案を求める。
  • RFQ：見積依頼。仕様が固まった後に価格や条件を求める。
