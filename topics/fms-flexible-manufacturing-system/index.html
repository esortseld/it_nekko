問11FMS

生産設備を結合して製造工程を柔軟に管理する

複数の工作機械や産業用ロボットを有機的に結合し、製造工程を柔軟に制御・管理する仕組みを整理します。

トップ / 学習要点 / 生産設備を結合して製造工程を柔軟に管理する
学習要点一覧へ戻る

1場面をつかむ

製造現場で、製品の種類や生産量が変わるたびに設備を大きく作り替えると、時間やコストがかかります。

そこで、工作機械、搬送装置、産業用ロボットなどを組み合わせ、工程全体を統括的に制御して、変化に柔軟に対応できるようにします。

2結論を先に押さえる

FMSは Flexible Manufacturing System の略で、複数の工作機械や産業用ロボットなどを有機的に結合し、生産プロセス全体を統括的に制御・管理する仕組みです。

製品の種類や生産量の変化に柔軟に対応するための仕組みとして押さえます。

3製造工程の構造を整理する

FMSは、製造現場の設備、ロボット、工程制御に注目する考え方です。

  • 設備：複数の工作機械や加工設備を組み合わせる。
  • ロボット：産業用ロボットや搬送装置などを結合する。
  • 制御：生産プロセス全体を統括的に制御・管理する。
  • 柔軟性：製品の種類や生産量の変化に対応する。

4試験で見る判断ポイント

試験では、複数の工作機械や産業用ロボットを結合し、製造工程を柔軟に制御・管理する説明からFMSを見分けます。

  • 対象が製造現場の設備やロボットであるか。
  • 工程全体を統括的に制御・管理しているか。
  • 製品種類や生産量の変化に柔軟に対応しているか。

5間違えやすい用語を分ける

  • SCM：調達から販売までの供給の流れを管理する考え方。
  • JIT：必要なものを必要なときに必要な分だけ供給する考え方。
  • リーン生産方式：生産現場のムダを減らす考え方。
  • FMS：工作機械やロボットを結合し、製造工程を柔軟に制御・管理する仕組み。

6まとめで定着させる

  • FMSは Flexible Manufacturing System の略である。
  • 工作機械や産業用ロボットなどを有機的に結合する。
  • 生産プロセス全体を統括的に制御・管理する。
  • 製品種類や生産量の変化に柔軟に対応する。
