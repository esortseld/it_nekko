問2ISO/IEC 27017

クラウドサービスの情報セキュリティ管理策を見分ける

クラウドサービスに関する情報セキュリティ管理策の規格を、品質、環境、個人情報保護の規格と区別して整理します。

トップ / 学習要点 / クラウドサービスの情報セキュリティ管理策を見分ける
学習要点一覧へ戻る

1場面をつかむ

クラウドサービスを利用するときは、利用者側だけでなく、サービス提供者側の情報管理や責任分担も考える必要があります。

ISO/IEC 27017は、このようなクラウドサービス固有の情報セキュリティ管理策を整理するための規格です。

2結論を先に押さえる

ISO/IEC 27017は、クラウドサービスに関する情報セキュリティ管理策の指針です。

試験では、規格番号そのものを丸暗記するよりも、「何を管理するための規格か」を分類して見分けます。

3規格の位置付けを整理する

ISO/IEC 27017は、ISMSの考え方を基礎に、クラウドサービスで必要になる管理策を扱います。

  • 対象：クラウドサービスに関する情報セキュリティ管理策。
  • 基礎：情報セキュリティマネジメントの考え方。
  • 視点：クラウドサービス提供者と利用者の役割や管理策。
  • 目的：クラウド利用時の情報セキュリティを適切に管理すること。

4判断ポイントを整理する

規格を見分けるときは、次の対応関係を押さえます。

  • ISO 9001：品質マネジメント。
  • ISO 14001：環境マネジメント。
  • JIS Q 15001：個人情報保護マネジメントシステム。
  • ISO/IEC 27017：クラウドサービスの情報セキュリティ管理策。

5間違えやすい考え方を分ける

  • ISO/IEC 27017を、品質や環境のマネジメント規格として読まない。
  • クラウドサービスの利用ルールだけでなく、情報セキュリティ管理策の規格として見る。
  • サイバーセキュリティ基本法のような法律・制度の論点とは分ける。

6まとめで定着させる

  • ISO/IEC 27017は、クラウドサービスの情報セキュリティ管理策に関する規格である。
  • ISMSの考え方を基礎に、クラウドサービス固有の管理策を扱う。
  • 試験では、「何を管理する規格か」を見分ける。
