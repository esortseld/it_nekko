令和8年度公開問題から探す

このページは、令和8年度ITパスポート試験公開問題の問番号から、対応する学習要点を探すための索引です。問題文は掲載せず、各問題で問われている知識を学習要点として整理します。

問1 / ストラテジ系 / 法務

学習要点：生成AI生成物と著作権

生成AIを使った生成物でも、既存著作物に依拠し、公開利用する場合は、著作権者の許諾が必要となる可能性があります。

学習要点を読む 問2 / ストラテジ系 / システム戦略・IT活用

学習要点：BYODとは何か

従業員の私物端末を、企業の許可と管理ルールのもとで業務に利用する考え方を整理します。

学習要点を読む 問3 / ストラテジ系 / 経営戦略

学習要点：マクシミン戦略とは何か

複数の戦略から、最悪の場合の利益が最も大きくなるものを選ぶ考え方を整理します。

学習要点を読む 問4 / ストラテジ系 / 企業活動・IT活用

学習要点：ハッカソンとは何か

開発者や企画者などが短期間でアイデアを出し合い、共同で成果物を作るイベントを整理します。

学習要点を読む 問5 / ストラテジ系 / 法務

学習要点：ビジネスモデル特許と特許法

インターネットを利用したビジネスの仕組みを、コンピュータシステムとして実現した場合に関係する法律を整理します。

学習要点を読む 問6 / ストラテジ系 / システム戦略・IT活用

学習要点：デジタルトランスフォーメーションとは何か

デジタル技術が、生活やビジネスに変化をもたらしていく考え方を整理します。

学習要点を読む 問7 / ストラテジ系 / 企業活動・経営戦略

学習要点：バリューエンジニアリングとは何か

製品やサービスの価値を、機能とコストの関係で捉える考え方を整理します。

学習要点を読む 問8 / ストラテジ系 / システム戦略・IT活用

学習要点：IoTを利用したシステムとは何か

モノに取り付けたセンサーなどから情報を集め、インターネット経由で活用する仕組みを整理します。

学習要点を読む 問9 / ストラテジ系 / 会計・財務

学習要点：間接費とは何か

学習要点を読む 問10 / ストラテジ系 / 企業活動

学習要点：ブレーンストーミングとは何か

会議で自由にアイデアを出し合うときの基本的な進め方を整理します。

学習要点を読む 問11 / ストラテジ系 / 法務

学習要点：パブリシティ権とは何か

有名人の氏名や肖像がもつ顧客吸引力を、無断で商業利用してよいかという観点で整理します。

学習要点を読む 問12 / ストラテジ系 / 企業活動・業務分析

学習要点：WFAとは何か

業務プロセスの流れを可視化し、課題抽出や改善検討に使う記述形式を整理します。

学習要点を読む 問13 / ストラテジ系 / 法務

学習要点：匿名加工情報とは何か

個人を識別できないように加工し、元の個人情報へ復元できないようにした情報を整理します。

学習要点を読む 問14 / ストラテジ系 / 企業活動

学習要点：CSRとは何か

企業が社会の一員として果たすべき責任を、ISO 26000の考え方とあわせて整理します。

学習要点を読む 問15 / ストラテジ系 / 経営戦略・マーケティング

学習要点：キャズムとは何か

新しい製品やサービスが普及するときに、初期市場から一般市場へ広がる際の大きな隔たりを整理します。

学習要点を読む 問16 / ストラテジ系 / AI活用

学習要点：アノテーションとは何か

教師あり学習で使うデータに、正解となる情報を付ける作業を整理します。

学習要点を読む 問17 / ストラテジ系 / AI活用・リスク管理

学習要点：AI利活用で注意すること

AIを業務で使うときに確認すべき、安全性、バイアス、差別、プライバシーの観点を整理します。

学習要点を読む 問18 / ストラテジ系 / 企業活動・生産管理

学習要点：コンカレントエンジニアリングとは何か

製品開発の複数工程を並行して進め、開発期間を短縮する考え方を整理します。

学習要点を読む 問19 / ストラテジ系 / システム戦略・ITサービス

学習要点：ASPとは何か

アプリケーションソフトウェアを、インターネット経由で利用するサービス形態を整理します。

学習要点を読む 問20 / ストラテジ系 / システム企画・調達

学習要点：RFPとは何か

システム導入や調達の際に、発注側が要求や条件を示して提案を求める文書を整理します。

学習要点を読む 問35 / マネジメント系 / ITガバナンス

学習要点：新しいシステムを入れるとき、経営陣は何をするのか

勤怠システムの入れ替えを例に、ITガバナンスにおける評価・指示・モニターを整理します。

学習要点を読む 問36 / マネジメント系 / プロジェクトマネジメント・契約管理

学習要点：契約に含まれないPCへ導入を求められたらどうするか

ソフトウェア導入作業を例に、契約範囲外の追加依頼、無償対応、発注者との合意、復旧判断を整理します。
